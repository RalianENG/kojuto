# Build-time deps for the sandbox image. setuptools + wheel are required
# so `pip install --no-build-isolation <sdist>` can compile wheels from
# setup.py / pyproject.toml — pip itself ships with the python:3.12-slim
# base, but those two are stripped from every modern Python image.
#
# Pinned to exact versions AND SHA256 hashes so the supply chain trust
# is the same as the digest-pinned base image. Updating either pin
# requires re-checking the hash:
#
#   pip download --no-deps --no-cache-dir -d /tmp/dl setuptools==<NEW> wheel==<NEW>
#   sha256sum /tmp/dl/*.whl
